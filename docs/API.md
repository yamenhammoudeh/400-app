## API Documentation for 400 Card Game

### Endpoints

#### Games

1. **Get Game**
   ```
   GET /api/games?id={gameId}
   ```

2. **Create Game**
   ```
   POST /api/games
   ```

3. **Update Game**
   ```
   PUT /api/games
   ```

4. **Delete Game**
   ```
   DELETE /api/games?id={gameId}
   ```

### Authentication

All API requests require proper authentication headers.

### Response Formats

Standardized JSON responses with proper status codes.
